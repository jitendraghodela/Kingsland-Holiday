A custom Elementor widget to display travel packages with detailed information.

== Description ==

The Kingsland Travel Package plugin provides Product Packages page and custom Elementor widget to display travel packages. It offers features to improve user experience and efficiency in showcasing travel packages, including displaying prices, discounts, hotel stars, and other relevant details.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/kingsland-travel-package` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the plugin by going to the Elementor editor and adding the "Kingsland Travel Package" widget to your desired section.

== Frequently Asked Questions ==

= How do I configure the plugin? =
Go to **Elementor > Widgets** in the WordPress admin panel to access and configure the plugin options.

= Is the plugin compatible with WordPress Multisite? =
Yes, this plugin is compatible with WordPress Multisite.
